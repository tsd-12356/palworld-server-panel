# Roadmap

This roadmap keeps the project focused on a simple promise: make Palworld dedicated server hosting easier, safer, and nicer to look at.

## Current Focus

- Stable Docker Compose one-click deployment.
- Reliable Ubuntu/Debian systemd installer.
- Clear save-slot workflows: backup, import, upload, switch, and delete.
- Pleasant glass-style UI with useful status feedback.

## Planned

- Add optional scheduled backups.
- Add a richer backup restore view with backup sizes and timestamps.
- Add optional panel login for users who want to expose it behind a reverse proxy.
- Add Docker host validation and clearer first-run diagnostics.
- Add more screenshots and short demo GIFs to the README.
- Add theme controls for users who prefer less animation.

## Later Ideas

- Multi-server management.
- Web-based audit log export.
- More Palworld config presets.
- Optional update notifications through webhook providers.
- More complete Docker mode test coverage.

## Not Planned For The First Stable Release

- Kubernetes deployment.
- Public SaaS hosting.
- Multi-user permission roles.
- Windows native service management.
