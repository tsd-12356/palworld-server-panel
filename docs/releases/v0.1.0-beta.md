# v0.1.0-beta

Initial public beta release of Palworld Server Panel.

## Highlights

- Modern light glass UI with dashboard screenshots and polished interactions.
- Docker Compose deployment with separate `panel` and `palworld` services.
- Ubuntu/Debian systemd installer for native VPS deployment.
- Server status, online players, logs, RCON, and visual config editor.
- Save-slot management: backup, upload zip import, create, switch, and delete.
- Manual update check and background update trigger.
- Installer wizard for environment checks, Palworld install, and repair actions.
- Audit trail and machine status overview.

## Known Notes

- Docker mode is implemented and should be treated as beta until tested on your target host.
- The panel does not include built-in login. Use Tailscale, ZeroTier, a trusted reverse proxy, or a private network.
- Docker mode mounts `/var/run/docker.sock` so the panel can control the Palworld container. Use only in trusted environments.

## Quick Start

```bash
git clone https://github.com/tsd-12356/palworld-server-panel.git
cd palworld-server-panel
cp .env.example .env
docker compose up -d --build
```
